Games に**シューティングゲームメーカー**を追加しました。

プログラミング不要で、パラメータ調整と敵の配置だけで独自のシューティングゲームを作れます。

[プレイヤー](/games/shooting-player) 

## 使い方

1. [エディタ](/games/shooting-editor) でステージを組む
   - ピアノロール風のUIで敵の出現タイミング・位置を配置
   - 敵タイプ（Scout / Sniper / Carrier / Swarm / Tank）ごとに移動・攻撃パターンを設定
2. 「プレイヤーで開く」ボタンで即プレイ
3. JSONをコピーして共有も可能

## 敵タイプ

| タイプ | 特徴 |
| --- | --- |
| Scout | 低耐久・直線移動の基本敵 |
| Sniper | 停止して自機を狙い撃つ |
| Carrier | アイテムをドロップする低速機 |
| Swarm | 波型移動でトリッキーに動く |
| Tank | 高耐久・広範囲に弾をばらまく |
